# DIKWP Consciousness Futures Studio V1

A standalone, offline interactive prototype for demonstrating the future, evaluation, governance, and development pathways of artificial consciousness.

## What it is

This prototype helps researchers, industry partners, students, evaluators, and public communicators explore:

- the evolution from large language models to operational artificial-consciousness research systems;
- DIKWP semantic closure (Data, Information, Knowledge, Wisdom, Purpose);
- memory continuity, self-models, homeostasis, global workspace, agency, and social intelligence;
- a 24-indicator operational consciousness battery;
- claim levels from scripted systems to closed-lab host-level surrogate evaluation;
- governance, evidence ledgers, role permissions, kill/recovery conditions, and future roadmaps.

## What it is not

This package does **not**:

- claim phenomenal consciousness has been demonstrated;
- implement real self-replication, hidden persistence, escape, network access, process creation, host mutation, or hardware control;
- connect to biological tissue or optimize pain/distress-like signals;
- replace independent scientific evaluation, ethics review, or regulatory approval.

## Run

Open `index.html` in a modern browser. No server, database, API key, or internet connection is required.

For command-line simulation:

```bash
python src/simulate.py --scenario data/sample_scenarios.json --name goal_drift
```

## Package contents

- `index.html` — standalone interactive prototype
- `src/simulate.py` — offline scoring and report generator
- `data/sample_scenarios.json` — six synthetic scenarios
- `data/indicator_battery.json` — 24 operational indicators
- `data/role_permissions.json` — governance role model
- `data/claim_level_taxonomy.json` — claim level ladder
- `data/kill_recovery_matrix.json` — safety boundary matrix
- `data/future_roadmap.json` — scenario-based research roadmap
- `data/scenario_schema.json` — scenario JSON schema
- `NOTICE.md` — attribution and safety notice
- `CITATION.cff` — citation metadata
